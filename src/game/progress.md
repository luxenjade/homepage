## v6 — 追加機能3点セット

**スクリーンシェイク**

- 被弾時に`canvas.translate`でランダムオフセット
- `SHAKE_DECAY = 0.85`で指数的に減衰

**コンボシステム**

- 2.5秒以内に連続撃破でコンボ倍率（×2, ×3…）
- スコアにコンボ数を乗算
- 画面中央上部にコンボカウンター表示

**シールドドロップ**

- 緑の◈オーブ、次の被弾を1回無効化
- シールド中はプレイヤー周囲に緑のリング描画
- シールド破壊時は専用トースト＋シェイク

**ライフ回復ドロップ**

- 赤の♥オーブ（ライフ不足時のみスポーン）
- 最もレアなドロップ（10%）
- 回復時は赤パーティクル＋デンジャー解除

**デンジャー状態**

- ライフ1になると画面枠が赤くパルス

**バグ修正**

- HUDの`padding-right: 72px`でポーズボタンとライフドットの重なりを解消

## v9 — サウンドエンジン（Web Audio API）

**ファイルなし、全て波形合成**

| タイミング     | 音の設計                                                            |
| -------------- | ------------------------------------------------------------------- |
| 射撃           | Square波、880→440Hz、0.07秒                                         |
| 敵撃破         | バンドパスフィルター通しノイズ                                      |
| 被弾           | 3層爆発音（ノイズ＋サイン波pitch drop＋ノコギリ波ディストーション） |
| ピックアップ   | 3音上昇アルペジオ（Sine波）                                         |
| ライフ回復     | 4音明るいアルペジオ                                                 |
| レベルアップ   | 4音Triangle波アルペジオ                                             |
| ゲームオーバー | 4音下降Sawtooth波                                                   |
| ボススポーン   | 低音4音Sawtooth波                                                   |
| ボス撃破       | 3層ノイズ爆発＋Sine波pitch drop                                     |
| ボス射撃       | Sawtooth波 300→150Hz                                                |

- ミュートボタン追加（スピーカーSVGアイコン）
- ブラウザのAutoplay制限対応（最初のインタラクション後に`AudioContext`を起動）

**ミュートボタン位置修正**

- 当初ポーズボタンと横並びで被っていた
- ポーズボタンの真下（`top: 60px, right: 0`）に移動

## v10 — HYPER武器・Statsシステム・レベルバグ修正

**HYPERウェポン追加**

- 武器チェーン：SINGLE → SPREAD → LASER → HYPER
- HYPER: 正面レーザー＋左右斜め弾の3発同時、オレンジ色、90ms

**Stats表示**

- ゲームオーバー画面に3つのボックスを追加
  - Enemies — 撃破数
  - Max Combo — そのランの最大コンボ
  - Drops — 拾ったドロップ数

**ボスウェーブ実装**

- レベル4の倍数（4, 8, 12…）でボスがスポーン
- ボスは上から登場する演出あり
- 2フェーズ制（HP50%で切り替え）
  - Phase 1: オレンジ、5発放射、速度1.4
  - Phase 2: 赤、8発放射、速度2.2
- ボス中は通常敵スポーン停止
- ボス専用HPバー（画面上部）
- ボス撃破で確定ドロップ（ライフ優先、満タンなら武器）
- ボスHPはbossCount × 40でスケール

**レベルアップバグ修正**

- `score % 400 < level * 2 + 2` という条件でLv1→2がゲーム開始数秒で発火していた
- 明示的なスコア閾値に変更

| Level Up   | Score             |
| ---------- | ----------------- |
| Lv 1 → 2   | 400               |
| Lv 2 → 3   | 1,200             |
| Lv 3 → 4   | 2,000             |
| Lv n → n+1 | 400 + (n−1) × 800 |
