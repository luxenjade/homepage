# 節8 医療制度とその活用

## 医療費と医療保険

私たちが医療機関で診察・治療を受けた際にかかる費用を⟦⟦医療費⟧⟧という。

経済的な理由で必要な医療を受けられないことがないよう、⟦⟦医療保険⟧⟧の制度がある。国民全体があらかじめ保険料を出し合って資金を蓄え、かかった医療費の多くはこの蓄えから支払う。

日本では⟦⟦1961⟧⟧年に⟦⟦国民皆保険⟧⟧を実現し、誰もが少ない負担で医療を受けられるようになっている。

### 窓口負担割合

| 対象                     | 負担割合         |
| ------------------------ | ---------------- |
| 義務教育就学後〜70歳未満 | ⟦⟦3⟧⟧割          |
| 保険証が提示できない場合 | ⟦⟦全額（10割）⟧⟧ |

公的医療保険では補われない部分を補うために⟦⟦民間保険⟧⟧もある（「私的保険」「任意保険」は誤称）。事前に必要性をよく検討する必要がある。

---

## 医療機関の種類

日本の医療機関は⟦⟦病院⟧⟧と⟦⟦診療所⟧⟧に大別できる。

| 区分       | 病床数                             |
| ---------- | ---------------------------------- |
| ⟦⟦病院⟧⟧   | 入院患者用ベッドが⟦⟦20床以上⟧⟧     |
| ⟦⟦診療所⟧⟧ | ⟦⟦20床未満⟧⟧またはベッドをもたない |

専門的な検査を受けるために大きな病院を受診するときは⟦⟦紹介状⟧⟧が必要（「招待状」「案内状」は誤り）。

### 医療機関の役割分担

| 医療機関                   | 役割                                                                 |
| -------------------------- | -------------------------------------------------------------------- |
| ⟦⟦かかりつけ医⟧⟧（家庭医） | 日常的な病気を診察し、個人・家庭の健康管理を担う                     |
| ⟦⟦地域医療支援病院⟧⟧       | 紹介された患者への医療提供でかかりつけ医を支援し地域医療の中核を担う |
| ⟦⟦特定機能病院⟧⟧           | 高度先進医療の提供・医療技術の開発・研修を行う                       |

## 患者の権利と医療への参加

### インフォームド・コンセント

医師などから十分な説明を受けた上で、納得して自ら治療方法を決めることを⟦⟦インフォームド・コンセント⟧⟧という。

医療関係者が患者に対して必要な情報を伝え、患者が納得した上で治療が行われることを指す。

### セカンド・オピニオン

医師の診断に確かめたいことがある場合に、別の医療機関の医師に意見を求めることを⟦⟦セカンド・オピニオン⟧⟧という。

### 臓器移植

病気の治療のために他の人の心臓・腎臓・肝臓などを移植することを⟦⟦臓器移植⟧⟧という。

- 本人が臓器提供の意思を示している場合に行われる
- **本人の意思のみで行うことができる**（家族の同意のみで行われるわけではない）
- 「家族の同意がある場合のみ行われる」は**誤り**

## 社会保険制度の全体像

| 保険制度     | 内容                       |
| ------------ | -------------------------- |
| ⟦⟦医療保険⟧⟧ | 病気・けがの医療費         |
| ⟦⟦年金保険⟧⟧ | 高齢になった時の給付       |
| ⟦⟦介護保険⟧⟧ | 介護が必要になった時の給付 |

## 献血

輸血や血液製剤を製造するために無償で血液を提供することを⟦⟦献血⟧⟧という。日本赤十字社が主体となって実施している。
