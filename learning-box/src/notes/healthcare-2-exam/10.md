# 節10 様々な保健活動や対策

## ヘルスプロモーション

私たち1人ひとりの健康課題は、生まれてから過ごしてきた地域や社会のあり様によって異なることから、日本では⟦⟦健康増進法⟧⟧の下に国民の健康づくりのための⟦⟦ヘルスプロモーション⟧⟧が行われている。

学校・地域・⟦⟦社会⟧⟧のレベルで行われる活動の調和をとること、そこで生活すれば無理なく自然と健康を保持増進できるように⟦⟦健康⟧⟧の視点で⟦⟦環境⟧⟧をデザインすることが大切である。

**⟦⟦すべての政策に健康の視点を⟧⟧**（Health in All Policies）：WHOが2008年に提唱。社会のあらゆる政策が健康の保持増進につながるようにすべきという方針。

### 仕掛け

つい健康的な行動をしてしまう・したくなるような環境の工夫を⟦⟦仕掛け⟧⟧という。例：立って仕事ができる机の導入など、働く⟦⟦環境⟧⟧を変えることで健康的な行動を選択しやすくする。

## 国際的な健康問題

⟦⟦グローバル化⟧⟧が進展した現在では、感染症・気候変動・⟦⟦自然災害⟧⟧など地球規模で影響し合う健康課題が増えている。そのため⟦⟦一国⟧⟧という単位ではなく、各国が協力して世界全体で解決に向けて取り組んでいる。

### 非感染性疾患（NCDs）

WHOは生活習慣の改善によって予防可能な疾患を⟦⟦NCDs（非感染性疾患）⟧⟧としてまとめている。

## 国際的な支援機関

### 政府機関・国連機関

| 機関               | 名称             | 略称       | 主な活動                            |
| ------------------ | ---------------- | ---------- | ----------------------------------- |
| 日本政府           | 政府開発援助     | ⟦⟦ODA⟧⟧    | 先進国→開発途上国への資金・技術提供 |
| 日本の独立行政法人 | ⟦⟦国際協力機構⟧⟧ | ⟦⟦JICA⟧⟧   | 日本のODAを包括的に実施             |
| 国連専門機関       | ⟦⟦世界保健機関⟧⟧ | ⟦⟦WHO⟧⟧    | 保健衛生に関する国際活動全般        |
| 国連機関           | ⟦⟦国連児童基金⟧⟧ | ⟦⟦UNICEF⟧⟧ | 子どもの栄養改善・疾病予防          |

> **JICAの略称の注意**：「JAICA」「GICA」は誤り。正しくは「JICA」。

JICAの取り組みにより世界に広まった「⟦⟦命のパスポート⟧⟧」：⟦⟦母子健康手帳⟧⟧

UNICEFの2023年報告：日本の⟦⟦5歳未満⟧⟧児死亡率（人口1000対）は⟦⟦2⟧⟧人（「1歳未満」「3歳未満」は誤り）

### WHOの活動内容

WHO活動の例：国際ガイドラインの策定、保健活動への技術支援、感染症対策、⟦⟦保健統計⟧⟧の作成

> **注意**：⟦⟦難民救護活動⟧⟧はWHOの活動には**含まれない**。難民救護は日本赤十字社・NGOの活動。

## 自転車活用のまちづくり

デンマーク・コペンハーゲン市の取り組み：市民の⟦⟦半数⟧⟧が自転車通勤・通学することを目指す。時速⟦⟦20⟧⟧kmで走行すると赤信号に当たらないよう信号を調整。

ヨーロッパ167都市加盟プロジェクト：市民の⟦⟦20⟧⟧%が自転車を主な移動手段にすると、年間の死亡者数が⟦⟦30000⟧⟧人程度減少すると試算

| 分野         | 期待される効果                                     | 懸念される影響                           |
| ------------ | -------------------------------------------------- | ---------------------------------------- |
| 健康         | 身体活動増加・肥満・糖尿病・心疾患・⟦⟦がん⟧⟧の減少 | 自転車事故増加・⟦⟦肺や気管支⟧⟧の疾患増加 |
| 都市環境     | ⟦⟦大気汚染⟧⟧・二酸化炭素排出の減少                 | 環境整備費用が必要                       |
| 財政         | 公的医療保険の支払い減少                           | —                                        |
| 産業         | ⟦⟦自転車⟧⟧関連産業の活性化                         | 自動車など他の交通産業が圧迫される       |
| まちの活性化 | 商店街の活性化・観光客増加                         | ⟦⟦放置自転車⟧⟧が増加する                 |
