# お知らせ

> 最終更新: 2026-04-23

現在配布している地理総合の定期試験過去問についてですが、今年度からの**地理総合担任教師の変更**により、定期試験での出題内容が**大幅に変更**される可能性があります。

過去問の試験範囲と実際の試験範囲にズレが生じることも考えられるので、当サイトでの過去問使用の際は実際の試験範囲をよく確認したうえの利用をおすすめします。
