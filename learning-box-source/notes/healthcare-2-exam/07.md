# 節7 保健制度とその活用

## 保健と医療の違い

健康を保つ様々な活動のことを⟦⟦保健⟧⟧という。これに対し、病気を診断・治療することを⟦⟦医療⟧⟧という。実際にはどこまでが健康でどこからが病気かの境界は曖昧であり、保健と医療は⟦⟦連続的⟧⟧に関係している。

---

## 保健行政機関

国および地方自治体が公共の責任において人々の健康を支える様々な活動を行うことを⟦⟦保健行政⟧⟧という。

### 行政の種類と担当省

| 行政の種類       | 対象                                 | 担当省庁                         |
| ---------------- | ------------------------------------ | -------------------------------- |
| ⟦⟦学校保健行政⟧⟧ | 児童・生徒・学生・教職員など学校生活 | ⟦⟦文部科学省⟧⟧                   |
| ⟦⟦一般保健行政⟧⟧ | 家庭（母子・高齢者）や地域社会       | ⟦⟦厚生労働省⟧⟧                   |
| ⟦⟦産業保健行政⟧⟧ | 職場（雇用労働者）の生活             | 労働基準監督署（⟦⟦厚生労働省⟧⟧） |
| ⟦⟦環境保健行政⟧⟧ | 公害対策・環境保全                   | ⟦⟦環境省⟧⟧                       |

### 保健センターと保健所

| 機関             | 設置場所               | サービスの特徴         |
| ---------------- | ---------------------- | ---------------------- |
| ⟦⟦保健センター⟧⟧ | 市区町村               | 住民に身近・地域密着型 |
| ⟦⟦保健所⟧⟧       | 主に都道府県・一部の市 | ⟦⟦広域的⟧⟧・専門的     |

> **注意**：「保健所は保健センターに比べ⟦⟦狭域的⟧⟧なサービスを行う」は**誤り**。保健所は**広域的**。

保健所が行う保健サービスの例：病院・飲食店の監視、感染症流行防止対策など。
保健所の配置スタッフ：保健師・看護師・⟦⟦医師⟧⟧・獣医師・薬剤師・食品衛生監視員など。

---

## 保健サービスの種類

| サービス       | 例                                             |
| -------------- | ---------------------------------------------- |
| ⟦⟦健康診査⟧⟧   | 妊婦健診・乳幼児健診（**地域**で行われるもの） |
| 健康診断       | 学校・職場で行われるもの                       |
| ⟦⟦健康教育⟧⟧   | 離乳食教室・介護予防教室                       |
| ⟦⟦予防接種⟧⟧   | インフルエンザ・BCGなど                        |
| ⟦⟦健康相談⟧⟧   | 保健師等の専門職が対応                         |
| ⟦⟦環境整備⟧⟧   | 人々の健康を促進する環境づくり                 |
| 健康情報の提供 | 行政機関HPでの統計資料・医療機関一覧など       |

> **注意**：⟦⟦医療保険⟧⟧や⟦⟦スポーツ施設の利用⟧⟧は保健サービスには**含まれない**。

---

## 予防の3段階

| 段階         | 内容                       | 例                         |
| ------------ | -------------------------- | -------------------------- |
| ⟦⟦一次予防⟧⟧ | 病気にならないようにする   | 予防接種・健康的な生活習慣 |
| ⟦⟦二次予防⟧⟧ | 病気を早期に発見・治療する | 健康診査・健康診断         |
| ⟦⟦三次予防⟧⟧ | 各機能を回復し社会復帰する | リハビリテーション         |
